# AI_Learning_Tracker
AI Learning Tracker

Description

AI Learning Tracker is a Core Python project that helps users track their learning progress. Users can add topics, update progress, delete topics, search topics, and view learning statistics. The project uses file handling to save data permanently.

Features

- Add Topic
- View Topics
- Update Progress
- Delete Topic
- Search Topic
- View Statistics
- Save Data in File

Technologies Used

- Python
- File Handling
- Functions
- Lists
- Dictionaries

How to Run

1. Download the project.
2. Open terminal in the project folder.
3. Run:

python main.py

Future Improvements
